# eav-vibe-coded

   Пример системы общего назначения, основанной на модели EAV (Entity-Attribute-Value)
   и интегрированной с локальной LLM deepseek-coder:6.7b для выполнения запросов на естественном языке
   Для помощи в написании кода использовалась та же LLM
   Использованные промпты сложены в папку prompts
   Первоначальная структура проекта получена из https://start.spring.io/ 
   
   Для сборки и выполнения приложения на компьютере должно быть установлено
   
   - Java 21
   - gradle 8.6
   - PostgreSQL 14.18
   - ollama 0.9.5
   - deepseek-coder:6.7b
